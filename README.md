#FreePasswordManager v.1.00
=================

A simple password manager that supports encryption.

#Usage:
=================
To run type:
```bash
python3 fpm.py
```

#Prerequisites:
=================
The following libraries / frameworks must be installed for FreePasswordManager to run:
- Python 3.x
- Qt 5.x framework
- PyQt5 
- pyperclip
